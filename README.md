# Python 随机密码生成器
 新手入门Python练手小项目
 ## 项目功能
 1. 自定义密码长度
 2. 自由选择是否包含大写字母、数字、特殊符号
 3. 自动生成高强度安全随机密码
 4. 带有输入错误容错处理，程序不会轻易崩溃
 ## 运行环境
 Python 3.x
 仅使用Python自带标准库，无需额外安装第三方模块
 ## 使用方法
 1. 运行 password.py 文件
 2. 根据控制台提示选择字符类型
 3. 输入密码长度，即可自动生成密码
 ## 代码知识点
 - random 随机模块
 - string 字符串内置模块
 - for 循环
 - if 判断语句
 - try-except 异常捕获
